# 日本語 Tracker

Трекер для изучения японского языка. Считает время занятий по разделам, сохраняет историю и показывает статистику для пользователей.

**→ [sahvea.github.io/nihongo](https://sahvea.github.io/nihongo/)**

---

## Функциональность

- **Таймер** — запуск/остановка с привязкой к разделу, переключение раздела без остановки таймера
- **Ручное добавление** — добавление времени в часах и минутах с заметкой
- **Разделы** — Кана, Кандзи, Anki, Грамматика, Чтение, Видео
- **Статистика** — плитка активности за 6 месяцев, распределение по разделам

## Стек

- React 19 + TypeScript + Vite
- Firebase Auth + Firestore
- CSS Modules
